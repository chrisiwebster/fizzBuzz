Test out the FizzBuzz function

Enter a number.

  • If it is divisible by 3, you will see 'Fizz'.
  • If it is divisible by 5, you will see 'Buzz'.
  • If it is divisible by 3 and 5, you will see 'FizzBuzz'.
  • If none of the above is true, you will see your number.

Give the counter a number

Enter a number. This function counts down from the number and checks each number against the FizzBuzz function above.
